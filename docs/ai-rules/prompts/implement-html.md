# Prompt — Implement HTML

## Objetivo

Transformar layout/mockup aprovado em HTML real com Tailwind/CSS mínimo.

## Template

```txt
Objetivo:
Implementar em HTML a seção [nome] aprovada visualmente.

Contexto:
O design já foi aprovado em mockup. Agora a tarefa é transformar em HTML limpo, responsivo e compatível com o projeto.

Arquivos envolvidos:
- [index.html ou arquivo da página]
- [CSS da página, se existir]
- docs/frontend/tailwind.md
- docs/layout/container.md
- docs/layout/sections.md

O que deve ser feito:
- criar HTML semântico
- usar Tailwind-first
- usar CSS complementar apenas se necessário
- respeitar container 1280px
- manter header/footer globais
- manter responsividade

O que não deve ser feito:
- não mudar copy aprovada
- não alterar componentes globais
- não usar valores aleatórios de container
- não criar CSS desnecessário
- não remover CSS antigo antes de screenshot after
```

---

## Regras obrigatórias

- Seguir `docs/ai-rules/global-rules.md`
- Seguir `docs/ai-rules/visual-rules.md`
- Seguir `docs/ai-rules/code-rules.md`
- Usar Tailwind-first
- Respeitar container 1280px
- Não alterar header/footer sem pedido explícito
- Não alterar outras páginas
- Não apagar CSS legado antes de validação

## Critérios gerais de aceite

- Entrega limitada ao escopo
- Visual consistente com NoPing
- Sem regressão aparente
- Responsivo considerado
- Código simples e reaproveitável
