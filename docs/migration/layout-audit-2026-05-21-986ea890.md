# Layout Audit — 2026-05-21

## Objetivo

Auditar o layout das páginas do site NoPing antes de qualquer alteração de código.

Esta auditoria deve verificar:

- uso de Tailwind;
- uso de CSS de página;
- regras de `100vh`, `100svh`, `min-height` e `height`;
- relação entre hero e header;
- comportamento do footer;
- uso de container;
- presença de valores legados como `1350px` ou `1360px`.

## Checklist por página

Para cada página, verificar:

### 1. Estrutura

- [ ] Usa header global?
- [ ] Usa footer global?
- [ ] Usa `<main>` corretamente?
- [ ] Usa `data-page`?
- [ ] Usa classe específica no body?

### 2. CSS/Tailwind

- [ ] Usa Tailwind?
- [ ] Usa CSS global?
- [ ] Usa CSS de componentes?
- [ ] Usa CSS de página?
- [ ] Usa CSS complementar?

### 3. Altura e viewport

Procurar por:

- [ ] `100vh`
- [ ] `100svh`
- [ ] `min-height`
- [ ] `height`
- [ ] `section-viewport`
- [ ] `hero`
- [ ] `calc(...)`
- [ ] `--header-height`
- [ ] `--np-header-height`

### 4. Header

- [ ] Header tem 72px?
- [ ] A página compensa o header?
- [ ] A compensação está no main?
- [ ] A compensação está na section?
- [ ] Existe compensação duplicada?

### 5. Footer

- [ ] Footer está no fluxo normal?
- [ ] Footer depende de margin manual?
- [ ] Footer é empurrado por `main flex: 1`?
- [ ] Footer sobrepõe conteúdo?
- [ ] Footer fica correto em página curta?

### 6. Container

Procurar por:

- [ ] `1280`
- [ ] `1350`
- [ ] `1360`
- [ ] `max-w-np-container`
- [ ] `max-width`
- [ ] `.container`
- [ ] `.layout-container`

### 7. Responsividade

- [ ] Existe comportamento mobile?
- [ ] Existe overflow horizontal?
- [ ] Cards quebram corretamente?
- [ ] Header/footer continuam corretos?

## Recomendações

### Recomendação 1 — Container

Padronizar todo conteúdo estrutural em:

```txt
1280px
```

Não usar:

```txt
1350px
1360px
```

---

### Recomendação 2 — Sections

Padronizar:

```txt
Header + Section = 100svh
```

com:

```css
min-height: calc(100svh - var(--np-header-height));
```

---

### Recomendação 3 — Footer

O footer deve funcionar por regra estrutural:

```css
body {
  min-height: 100svh;
  display: flex;
  flex-direction: column;
}

main {
  flex: 1;
}

footer {
  flex-shrink: 0;
}
```
